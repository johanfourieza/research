# research

Replication data and reproducible files for research projects by
**Johan Fourie** (Stellenbosch University) and collaborators.

Each subfolder contains a self-contained data package for one project, with
its own README, codebook, license and loader script.

## Projects

| Folder | Project | Paper |
|---|---|---|
| [`2026_voortrekker/`](2026_voortrekker/) | Demographic Pressure, Emancipation and Selection into the Great Trek | Fourie & Links (2026), *European Review of Economic History* |
